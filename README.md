# SmartRepair API

## Project Name

SmartRepair API Computer Repair Booking and Ticket Management System

## Project Description

SmartRepair API is a web application project for managing computer repair bookings, customer repair tickets, technicians, services, and appointments.

## Group Members

* Student 1: Database Designer
* Student 2: Backend Planner
* Student 3: Frontend Designer

## Technologies

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, TypeScript, Express
* Database: MySQL
* Tools: GitHub and Jira

## Main Features

* Customer management
* Repair ticket tracking
* Technician assignment
* Service listing
* Appointment booking

## Planned API Routes

* GET /customers
* GET /technicians
* GET /services
* GET /tickets
* GET /appointments

## Milestone 1

Repository setup, project idea, group members, GitHub, and Jira planning.

## Milestone 2

Database design, ERD, frontend wireframes, backend route planning, and team responsibilities.

## Milestone 3

Backend implementation using TypeScript, Express, MySQL, db.ts, and route files.
